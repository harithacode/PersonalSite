Selected Projects and Skills
Delanuay Triangulation

This is a triangulation algorithm from my Computational Geometry course. I had selected it for my project mainly because of it's aesthetic appeal. It my favorite project so far. The output contains a triangulated mesh overlayed on the original picture. (Sample output). This project taught me a lesson about unit testing functions before using them. This algorithm relies on accurate floating point calculations. Initially it gave a hap-hazard mesh. This was caused by a buggy function that was supposed to be doing accurate floating point division but was not. Fixed by: "from __future__ import division"

Information flow on Twitter

In fall of 2014 I worked for Dr. Gang Han from Greenlee School of Journalism to do some analysis on how users shared information over Twitter. Specifically we looked at diffusion patterns of URLs. For this purpose, I prepared a list of users sharing a viral URL and see how they relate to each other. We experimented with different relation criteria like following relationship, replies, user mentions and retweets. I had also parsed huge text files with tweets for the purpose feeding them to the database (Link to the python script). This project gave me some fun insights about how Twitter users connect.
